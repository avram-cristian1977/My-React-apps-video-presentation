React Mr Meme

This is a simple meme app that can teach you to create your own meme  (cropping, trimming). Fast and free.
Also the app consumes a very reach memes api with search by keyword inplemented and copy to clipboard functionality
and therefore you can paste everywhere the meme's url.
The app has routes, dynamic routes (detail route / detail api), 
authentication with Firebase Authentication (sign up firebase api and sign in firebase api)
localStorage item. (app survives refresh)

I used hooks like useContext, useState, useRef, useEffect. Firebase Authentication.
